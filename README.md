# Gatepass System

This project is a Gatepass Management System developed to manage student gatepass requests.

## Features
- Student login
- Gatepass request
- Admin approval
- Secure authentication

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- MongoDB

## How to Run the Project
1. Install Node.js
2. Open the project folder
3. Run:
   npm install
4. Start the server:
   npm start
